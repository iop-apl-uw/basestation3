# Basestation3 documents 

## gliderdac

Documentation on the basestation extension for generating netCDF files suitable for submission to the [IOOS GliderDAC](https://ioos.noaa.gov/project/underwater-gliders/) can be found [gliderdac/gliderdac.md](gliderdac/gliderdac.md)

## vis visualization server

Documentation on the basestation visualization server - vis - is in [vis.txt](vis.txt)
A sample mission.yml file for vis - with detailed docs in the comments - is in [missions.yml](missions.yml)

## Whole Mission Timeseries/Profile and parquet file contents

The whole mission time series netcdf files (`--make_mission_timeseries`) and whole mission profile (`--make_mission_profiles`) and parquet files (`--generate_parquet`) contents can be extended using a `--whole_mission_config` file [whole_mission.yml](whole_mission.yml).

## FlightModel

The Flight Model System (FMS) is documented in [FlightModel.pdf](FlightModel.pdf?raw=true)

## Basestation QC system

The quality control system for data in basestation3 processing is described in [Seaglider_Quality_Control_Manual.html](https://htmlpreview.github.io/?https://github.com/iop-apl-uw/basestation3/blob/master/docs/Seaglider_Quality_Control_Manual.html)

## RBR tridente naming conventions

Details on the [RBR Tridente](https://rbr-global.com/products/sensors/rbrtridente/) naming convention used in the basestation are in [tridente.md](tridente.md)
