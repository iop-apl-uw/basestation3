codaTODO Oxygen Plot Help

This plot shows dissolved oxygen vs. depth from a codaTODO oxygen sensor. The title states whether the data shown has been corrected for salinity and depth, and whether it has been restricted to QC_GOOD points only.

  • Corrected O2 traces (top x-axis, umol/kg) are the basestation-corrected dissolved oxygen values, dive and climb shown separately. If no correction was available, the plot instead shows the uncorrected instrument value with the title noting "not corrected".
  • Sat O2 traces show the instrument's own oxygen saturation percentage output, uncorrected.
  • Temperature traces (bottom x-axis, C, hidden by default - toggle on via the legend) show the oxygen sensor's own temperature channel, for comparing against the CTD temperature.
  • Shaded/overlay regions (when present) mark sample timeouts and the commanded sampling range for the instrument, same convention as other science plots - see the sampling traces help.
