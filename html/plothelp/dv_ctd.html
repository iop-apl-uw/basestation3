CTD Plot Help

There are two versions of this plots - non-RAW plots which show the correct CTD data, and RAW plots which show the uncorrected CTD data for Seabird CTDs. RBR Legatos have a different plot for RAW data (data as reported by the instrument).

All science traces on this plot report the data values, depth, time into the dive (in minutes) and the index of the data value in the dive in the hovertips. For scicon based instruments, the data index is for the combined profiles (dive, loiter, climb, surface loiter).

Salinity

  • For non-RAW plots, Salinty traces are the Basestation corrected salinity - they are taken from the salinity netCDF variable
  • For RAW plots Salinity traces are taken from the salinity_raw netCDF variable and represent the uncorrect salinity values


Temperature

  • For non-RAW plots, Temperature traces are the basestation corrected temperature - they are taken from the temperature netCDF variable
  • For RAW plots Temperature traces are taken from the temperature_raw netCDF variable and represent the uncorrect temperature values


Conductivity

  • For non-RAW plots, Conductivity traces are the basestation corrected conductivity - they are taken from the conductivity netCDF variable
  • For RAW plots Conductivity traces are taken from the conductivity_raw netCDF variable and represent the uncorrect conductivity values


Sigma-T

For non-RAW plots, sigma_t are the basestation calculated στ and is taken from the sigma_t netCDF variable. This trace is not tied to a x-axis - use the hovertips to interpret the values.



Buoyancy Frequency

For non-RAW plots, Buoyancy Frequency are the basestation calculated Buoyancy Frequency (N). The value is calculated for the plot only and is not persisted into the netCDF file. Unlike the traditional radians per second, this trace is in cycles per hour. This trace is not tied to a x-axis - use the hovertips to interpret the values.
