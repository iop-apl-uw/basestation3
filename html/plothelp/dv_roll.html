Roll Control Plot Help

This plot is used to check/re-derive the roll trim center (C_ROLL_DIVE, C_ROLL_CLIMB) by comparing observed vehicle roll angle against roll motor control counts. Dive and climb are fit and plotted separately, since the roll center can differ between them.

  • Observed roll - dive/climb points are actual roll angle vs. roll control counts seen during that segment.
  • Current center - dive/climb marks the roll control value implied by the glider's currently commanded C_ROLL_DIVE/C_ROLL_CLIMB.
  • Linear model - dive/climb is a linear fit to this dive's data, reporting its own implied roll center.

Where the linear model's zero-crossing (implied center) differs noticeably from the current center, that's a signal the roll trim parameter should be updated.
