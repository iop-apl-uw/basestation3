Vertical Velocity Plot Help

This plot compares observed vertical speed against the hydrodynamic flight model's (HDM) predicted speed vs. depth, and is used to check the vehicle's buoyancy/VBD (C_VBD, volmax) trim.

  • Vert Speed dz/dt is the observed vertical speed from the pressure sensor.
  • Vert Speed Desired (dive/climb) is the commanded target speed.
  • Uncorrected model w/glider parms is the flight model's prediction using the currently commanded trim parameters, with no bias correction.
  • Vert Speed Buoy/Pitch (current FMS-HDM) and the biased FMS-HDM variant show the flight model output before/after searching for a buoyancy bias that best matches the observed speed.
  • Upwelling Descent/Upwelling Ascent (when shown) is the ocean vertical velocity implied by the residual between observed and modeled glider speed.

The x-axis subtitle reports the best-fit buoyancy/VBD bias found for this dive and the trim parameters (C_VBD, volmax, MAX_BUOY, SM_CC) it implies, alongside the vehicle's currently commanded settings - large discrepancies indicate the trim parameters should be updated.
