TS Plot Help

This plot displays Temperature vs Salinity, color coded by depth. Additionally, density contour lines are plotted. There is a non-RAW and RAW version of this plot, with the RAW version is plotted if the --plot_ts_raw is set. For gliders with $USE_ICE set to a non-zero value, the freezing point of seawater is plotted

All science traces on this plot report the data values, depth, time into the dive (in minutes) and the index of the data value in the dive in the hovertips. For scicon based instruments, the data index is for the combined profiles (dive, loiter, climb, surface loiter).



Dive Climb

The Dive and Climb traces are plot the temperature vs. the salinity netCDF variables. If the plot is a RAW plot, then temperature_raw and salinity_raw are used.



Freeze Point

The Freeze Point traces are calculated from salinity (or salinity_raw for RAW plots), ctd_pressure, avg_latitude and avg_longitude netCDF variables.
