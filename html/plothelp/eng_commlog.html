CommLog Counter Values Plot Help

This plot tracks the "counter line" telemetry reported at the end of every surfacing in comm.log, across the whole mission vs. call number (the top axis maps calls to dive number).

  • Pitch, Depth, Internal Pressure, Relative Humidity, Internal Temperature, and battery voltages are all shown, most scaled so they share the left AD value axis (the scale factor is in each trace's legend label); a few use the right value axis directly.
  • <motor> position traces show VBD/pitch/roll motor position at the end of each surfacing.
  • CTD traces (when present) show CTD readings taken during the surface interval.

This is a coarse, mission-wide sanity check on vehicle health (e.g. slowly rising internal humidity/pressure, or drifting battery voltage) using only what's logged every surfacing - it doesn't require processed per-dive netCDF files.
