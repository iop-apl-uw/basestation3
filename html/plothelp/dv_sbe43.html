SBE43 Oxygen Plot Help

This plot shows dissolved oxygen vs. depth from a Seabird SBE43 oxygen sensor. The title states whether the data has been corrected for salinity and depth, whether it's restricted to QC_GOOD points, and the depth binning if the data is a binned profile.

  • Corrected O2 traces (top axis, umol/kg) are the basestation-corrected dissolved oxygen, dive and climb shown separately. If no correction was available, the uncorrected value is shown instead and the title notes "not corrected".
  • Sat O2 traces show the instrument's own oxygen saturation percentage output, uncorrected.

The annotation below the plot (when present) reports the instrument calibration comment.
