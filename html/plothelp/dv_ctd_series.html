CTD Series Plot Help

This is an animated version of the CTD plot, stepping through a rolling window of recent dives (controlled by the ctd_series_divesback option) rather than showing a single dive. It is only generated when --enable_ctd_series is set.

Use the Play/Pause buttons or drag the Dive: slider below the plot to step through dives. Each frame shows the same salinity/temperature/conductivity traces vs. depth as the single-dive CTD plot, for that dive.

As with the single-dive plot, a separate RAW version is generated for Seabird CTDs (uncorrected data) alongside the corrected version, unless the glider carries an RBR Legato or --plot_ctd_raw is set to skip it.
