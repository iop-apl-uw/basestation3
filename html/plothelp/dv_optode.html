Optode Oxygen Plot Help

This plot shows dissolved oxygen vs. depth from an Aanderaa optode (3830/4330/4831, whichever is installed - shown in the output file name). The title states whether the data has been corrected, whether it's restricted to QC_GOOD points, and the depth binning if the data is a binned profile.

  • Corrected O2 (top x-axis, umol/kg) is the basestation-corrected dissolved oxygen, taken from the aanderaa<type>_dissolved_oxygen netCDF variable. If no correction was available, only the instrument-reported trace below is shown and the title notes "not corrected".
  • Instrument Reported O2 is the raw, uncorrected value as reported by the optode itself, shown alongside the corrected trace for comparison.
  • Instrument Reported Temp (bottom x-axis, C, when present) is the optode's own temperature channel.
  • Sat O2 (when present) is the instrument's oxygen saturation percentage output.

The annotation below the plot reports the instrument calibration comment, drift gain (if set), and count of sample timeouts, when available.

Shaded/overlay regions (when present) mark sample timeouts and the commanded sampling range - see the sampling traces help.
