Mission Map Plot Help

This is a static map of the glider's full mission track, plotted over bathymetry.

  • The light salmon track is the glider's GPS-fix track over the whole mission.
  • Red + markers highlight recent surfacings.
  • Shading is ocean depth (bathymetry), darker blue is deeper.

Unlike the other plots in this reference, the mission map is rendered as a static image rather than an interactive plot, so it has no hover tooltips or zoom/pan controls.
