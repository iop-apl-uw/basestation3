Course Through Water Plot Help

This plot shows the glider's track relative to the water (Course Through Water, computed from the flight/hydrodynamic model) alongside its track relative to the ground (Course Over Ground, which includes the effect of ocean current), in meters of northward/eastward displacement from the start of the dive.

  • HDM traces use the basestation's Hydrodynamic Model.
  • GSM traces use the Gliderscope/steady-state model; these are shown by default when no CTD-timebase HDM solution is available for the dive.
  • AD2CP traces (when present) use velocities measured directly by an AD2CP current profiler rather than a flight model - both the through-water and (adding the profiler's measured ocean velocity) over-ground versions are shown.
  • GPS2/GPSEND mark the surface GPS fixes bracketing the dive.
  • The colored line segments radiating from the origin show the commanded heading and the +/- heading error band (log_HEAD_ERRBAND) used by the pilot's steering target.
