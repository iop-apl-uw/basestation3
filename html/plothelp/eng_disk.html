Disk Usage Plot Help

This plot tracks onboard storage (SD card and any other logger with free-space reporting) over the mission, vs. dive number.

  • SD free kB and any other <sensor> free-space traces (left axis) show remaining free space, one trace per logger/device that reports it.
  • SD file count and SD dir count (right axis) track the number of files and directories on the SD card.

The x-axis subtitle projects, from the recent free-space trend, how many dives/days remain before each device fills up and an estimated end date. A steepening slope or an early projected fill date is a signal to trim logged file types/rates for the rest of the mission.
