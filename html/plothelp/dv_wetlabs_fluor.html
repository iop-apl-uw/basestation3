WETLabs Fluorescence Plot Help

This plot shows one fluorescence channel (e.g. chlorophyll, CDOM - named in the title) from a WETLabs puck (wlbb2fl, wlbbfl2, wlbb3, or wlfl3, shown in the output file name) vs. depth, dive and climb traces shown separately. A separate plot is generated per fluorescence channel the instrument reports, preferring the basestation-adjusted value over the raw instrument value when both are available.

The annotation below the plot (when present) reports the instrument calibration comment.

See the WETLabs backscatter plot for the instrument's optical backscatter channels.
