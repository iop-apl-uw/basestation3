Vertical Velocity (New Model) Plot Help

This plot compares observed vertical speed against several flight-model predictions vs. depth, and is used to check the vehicle's buoyancy/VBD and hydrodynamic drag trim.

  • Vert Speed observed from pressure (solid black line) is the measured vertical speed - the reference all model traces are compared against.
  • Vert Speed Desired (dive/climb) is the commanded target speed.
  • Model as flying on glider uses the currently commanded C_VBD and hydrodynamic drag coefficients (HD_A, HD_B), no correction applied.
  • Buoyancy only corrected model applies just a buoyancy/C_VBD bias fit to this dive's data.
  • Additional model traces (when present) come from the full hydrodynamic regression fit - see the regression fit plot for that fit's own diagnostics.

Each model trace's legend label reports its implied parameters and RMS error (model minus observed vertical speed) - lower RMS is a better-fitting model.
